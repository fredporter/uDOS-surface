# VA2 boundary (not VA1)

VA1 is **TypeScript only** (`do` in this package). The following are **VA2** or later:

- **Go tools:** `ucode-cli`, `ucoin-ledger`, `meshcore-bridge`, game containers, ComfyUI bridge, …
- **TUI:** Bubble Tea or any full-screen terminal UI
- **Commands (examples):** `do run`, `do check`, `do fmt` (uCode), `do tui`, `do ucoin`, `do device scan`, `do pkg install`, `do world`, `do portal`, `do trade`, `do mp`

Repo stubs under `tools/` remain placeholders until VA2 ships; **VA1 must not spawn them** from the `do` CLI.
